# Social Icons Blueprint Documentation

## Identity
- **System Name**: `social-icons`
- **Purpose**: A comprehensive social media connectivity suite.
- **Behaviors**: 
    - Supports 15+ major platforms (Facebook, Twitter, Instagram, etc.).
    - Click-to-call and Email obfuscation support.
    - Responsive labels for Desktop/Mobile.

## Admin Settings (Blueprints)
- **Platform Groups**: Each platform (e.g., `social_facebook_group`) contains a URL field and an enabled toggle.
- **Show Names on Mobile**: `social_show_labels_mobile` (Toggle)
- **Show Names on Desktop**: `social_show_labels_desktop` (Toggle)

## UX/UI
- **Classes**: `.social-icons-list`
- **Icons**: Uses LineAwesome and SVG alternates.
