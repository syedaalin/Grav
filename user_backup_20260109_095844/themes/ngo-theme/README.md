# NGO Theme (2026 Agentic Standard)

![NGO Theme](screenshot.webp)

**NGO Theme** is a state-of-the-art Grav CMS theme designed for non-profit organizations, religious institutions, and businesses seeking a premium, high-performance digital presence. Built on the **2026 Agentic & Premium Standards**, it combines "Nature Distilled" aesthetics with cutting-edge functional logic.

## üõ† Installation

### GPM Installation (Recommended)
From the root of your Grav install, run:
```bash
bin/gpm install ngo-theme
```

### Manual Installation
1. Download the zip from GitHub.
2. Unzip into `user/themes/ngo-theme`.
3. Set `theme: ngo-theme` in your `user/config/system.yaml`.
